# CSS Animation
❔ Create a animation

```

  animation-name: moveInLeft;
  animation-duration: 3s;
}
```
```
@keyframes moveInleft{
  0%{
    opacity: 0;
  }
  
  80%{
    opacity: 80;
  }
  
  100%{
    opacity: 100%;
  }
}
```
